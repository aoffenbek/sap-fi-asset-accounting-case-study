# Depreciation Run

## Transaction Code

AFAB – Depreciation Run

## Purpose

Calculate and post periodic depreciation for fixed assets.

## Example

Asset Value: 1500 EUR
Useful Life: 3 Years

Annual Depreciation: 500 EUR
Monthly Depreciation: 41.67 EUR

## Accounting Entry

Debit: Depreciation Expense
Credit: Accumulated Depreciation
